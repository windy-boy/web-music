### web-music

* 仿网易云音乐，闲暇之余所作，查漏补缺，不定期更新。
* 接口由[free-api](https://www.free-api.com/doc/399)提供

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
